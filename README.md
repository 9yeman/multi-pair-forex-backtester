# Multi-Pair Forex Backtester

This project processes Forex data from MetaTrader, cleans it, generates multiple timeframes, 
and performs backtesting on several trading strategies. Results can be used to train a trading bot.

---

Bu proje, MetaTrader'dan alınan Forex verilerini işleyip temizler, farklı zaman dilimleri üretir 
ve çeşitli stratejiler üzerinde backtesting yapar. Sonuçlar daha sonra bir ticaret botunu eğitmek için kullanılabilir.
